C'est-à-dire que la distance de chaque sommet au centre de gravité sera égal à __1__. # Le tétraèdre régulier Plaçons le sommet `A` en `(0,1,0)` et `B` dans le plan XY. XY est, par ailleurs, un plan de symétrie pour le tétraèdre. On obtient les coordonnées suivantes : * A : (0, 1, 0) * B : (B_x, B_y, 0) * C : (C_x, B_y, C_z) * D : (C_x, B_y, -C_z) Il reste 4 inconnues que l'on peut trouver en remarquant que les arrêtes ont toutes la même longueur : * AB = AC = AD = BC = BD = CD * AB = B_x^2 + {(1 - B_y)}^2 * AC = C_x^2 + {(1 - B_y)}^2 + C_z^2
